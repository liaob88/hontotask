# hontotask
## What is this?
hontotask creates todosit tasks about a book for me when I enter the number of chapters and sections in the book. "hon" means book in Japanese.

## Examples
If I enter the book information like below,

- title
  - hontotask-book
- chapters count
  - 2
- sections count
  - chapter1: 3 sections
  - chapter2: 2 section
